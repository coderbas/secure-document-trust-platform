# Secure Document Trust Platform

## Overview

The Secure Document Trust Platform is a Flask-based cryptographic application that provides:

- RSA Digital Signatures
- Signature Verification
- AES-256 File Encryption
- AES-256 File Decryption
- Audit Logging
- Dashboard Analytics

## Security Objectives

### Confidentiality
AES-256-GCM Encryption

### Integrity
RSA Digital Signatures

### Authentication
Public Key Verification

### Non-Repudiation
Digital Signature Validation

## Technologies

- Python
- Flask
- SQLite
- Cryptography Library
- Bootstrap

## Features

- Sign Documents
- Verify Signatures
- Detect Tampering
- Encrypt Files
- Decrypt Files
- Audit Logs
- Dashboard Statistics
